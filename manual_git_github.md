# Manual para configurar el git por ssh

## Comando para crear la clave en el ordenador 

ssh-keygen -t ed25519 -C"balderascarmonaivan@gmail.com"

## Comando para comprobar si tienes la contraseña

ls -la ~/.ssh

y debajo tiene que salir unos archivos que terminen o se llamen asi: id_ed25519 y id_ed25519.pub
